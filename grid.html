CSS Grid

O que é CSS Grid?

CSS Grid é um sistema de layout baseado em grades que permite organizar elementos em colunas e linhas de maneira flexível.

Principais propriedades:

  • grid-template-columns: Define as colunas da grade.
  • grid-template-rows: Define as linhas da grade.
  • gap: Controla o espaçamento entre os elementos.

O que são os fr (frações)?

O fr é uma unidade no CSS Grid que representa uma fração do espaço disponível. Se você tem 1fr, significa que o item ocupa uma fração do espaço restante. Se você tiver 1fr 2fr, a primeira coluna ocupará 1 parte do espaço, e a segunda coluna ocupará 2 partes, dividindo o espaço restante de maneira proporcional.

Exemplo: Se você definir grid-template-columns: 1fr 2fr 1fr;, o espaço da grade será dividido em 4 partes iguais. A primeira e a terceira colunas ocuparão 1 parte cada, enquanto a segunda coluna ocupará 2 partes do espaço.
